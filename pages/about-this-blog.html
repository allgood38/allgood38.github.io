About this blog

This is a blog about the technologies I encounter on a daily basis, whether it be through the Computer Engineering program, or my own interests. My main motivation is to provide information to solve problems I have encountered where there is no trivial solution already posted online.

Currently I have published no real useful content, I'm still working on getting used to writing less verbose material, with more useful code snippets for people to use.

I have an account on GitHub where update projects I'm interested it, right now my main focus is on a design team at Queen's University to build an autonomous sailboat. Check out the team's website for more info.
